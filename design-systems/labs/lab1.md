<!-- Example Section -->
<div class="example-section">
  <div class="container">
    <div class="example-content animate-fade-in-up">
      <h2 class="example-title">Глава третья: Практический пример</h2>
      <p class="example-description">
        Рассмотрим реальный пример анализа предметной области 
        библиотечной информационной системы
      </p>
      
      <div class="chapter-highlight">
        Информационная система для управления библиотекой и электронным каталогом
      </div>
      
      <p class="chapter-text">
        Предметная область данного проекта — это сфера библиотечного обслуживания, 
        а именно деятельность библиотеки, совмещающей в себе фонд хранения печатных 
        и электронных материалов, читальные залы, а также предоставление доступа 
        к каталогу через онлайн-портал. Система предназначена для комплексной 
        автоматизации всех ключевых процессов библиотеки. Цель — оптимизировать 
        работу сотрудников, улучшить обслуживание читателей и обеспечить 
        эффективный учет библиотечного фонда.
      </p>
      
      <h3 style="color: #000; font-size: 1.5rem; margin: 2rem 0 1rem 0; text-align: center;">Ключевые Термины</h3>
      
      <ul class="analysis-card-list" style="text-align: left; max-width: 800px; margin: 0 auto;">
        <li><strong>Библиотекарь</strong> — сотрудник библиотеки, пользователь с правами администратора системы, отвечающий за учет фонда и обслуживание читателей.</li>
        <li><strong>Читатель</strong> — зарегистрированный пользователь библиотеки, осуществляющий поиск и бронирование материалов.</li>
        <li><strong>Экземпляр</strong> — физическая или электронная единица хранения в библиотечном фонде (книга, журнал, CD и т.д.).</li>
        <li><strong>Выдача</strong> — оформленный запрос на получение читателем одного или нескольких экземпляров. Имеет статусы (например, «Забронирован», «Выдан», «Продлен», «Возвращен», «Просрочен»).</li>
      </ul>
      
      <h3 style="color: #000; font-size: 1.5rem; margin: 2rem 0 1rem 0; text-align: center;">Основной Функционал Приложения</h3>
      
      <p class="chapter-text">
        Система состоит из двух основных модулей: административная панель 
        (для Библиотекаря) и публичный каталог (для Читателей).
      </p>
      
      <h4 style="color: #000; font-size: 1.3rem; margin: 1.5rem 0 1rem 0; text-align: center;">А. Административная Панель Управления</h4>
      
      <h5 style="color: #000; font-size: 1.1rem; margin: 1rem 0 0.5rem 0; text-align: left;">Управление Библиотечным Фондом:</h5>
      <ul class="analysis-card-list" style="text-align: left; max-width: 800px; margin: 0 auto;">
        <li>Ведение базы данных экземпляров: добавление, редактирование и списание.</li>
        <li>Указание библиографических данных, классификации, ключевых слов.</li>
        <li>Управление статусом экземпляров (доступен, выдан, в обработке, списан).</li>
        <li>Возможность временного снятия экземпляра с публичного доступа.</li>
      </ul>
      
      <h5 style="color: #000; font-size: 1.1rem; margin: 1rem 0 0.5rem 0; text-align: left;">Учет Фонда:</h5>
      <ul class="analysis-card-list" style="text-align: left; max-width: 800px; margin: 0 auto;">
        <li>Ведение учета поступлений новых экземпляров с указанием источника и даты.</li>
        <li>Отслеживание движения экземпляров: система автоматически меняет статус при выдаче и возврате.</li>
        <li>Формирование отчетов о движении фонда за период, статистика популярности.</li>
      </ul>
      
      <h5 style="color: #000; font-size: 1.1rem; margin: 1rem 0 0.5rem 0; text-align: left;">Управление Выдачами:</h5>
      <ul class="analysis-card-list" style="text-align: left; max-width: 800px; margin: 0 auto;">
        <li>Централизованная панель просмотра всех текущих и архивных выдач.</li>
        <li>Изменение статусов выдач с уведомлением читателя (например, о готовности к выдаче или просрочке).</li>
        <li>Просмотр истории выдач по читателям и экземплярам.</li>
      </ul>
      
      <h4 style="color: #000; font-size: 1.3rem; margin: 1.5rem 0 1rem 0; text-align: center;">Б. Публичный Электронный Каталог</h4>
      
      <h5 style="color: #000; font-size: 1.1rem; margin: 1rem 0 0.5rem 0; text-align: left;">Поиск по Каталогу:</h5>
      <ul class="analysis-card-list" style="text-align: left; max-width: 800px; margin: 0 auto;">
        <li>Просмотр экземпляров с фильтрацией по жанрам, автору, году издания и тегам.</li>
        <li>Расширенный поиск по названию, аннотации, ключевым словам.</li>
      </ul>
      
      <h5 style="color: #000; font-size: 1.1rem; margin: 1rem 0 0.5rem 0; text-align: left;">Бронирование и Запрос:</h5>
      <ul class="analysis-card-list" style="text-align: left; max-width: 800px; margin: 0 auto;">
        <li>Стандартный процесс: поиск, проверка доступности, оформление запроса на выдачу.</li>
        <li>Уведомление о готовности заказа и сроках возврата.</li>
      </ul>
      
      <h5 style="color: #000; font-size: 1.1rem; margin: 1rem 0 0.5rem 0; text-align: left;">Личный Кабинет Читателя:</h5>
      <ul class="analysis-card-list" style="text-align: left; max-width: 800px; margin: 0 auto;">
        <li>История текущих и прошлых выдач.</li>
        <li>Отслеживание статуса запросов и управление продлением.</li>
        <li>Просмотр начисленных штрафов за просрочку.</li>
      </ul>
      
      <h5 style="color: #000; font-size: 1.1rem; margin: 1rem 0 0.5rem 0; text-align: left;">Информационные Разделы:</h5>
      <ul class="analysis-card-list" style="text-align: left; max-width: 800px; margin: 0 auto;">
        <li>«О Библиотеке», «Режим работы», «Правила пользования», «Контакты».</li>
      </ul>
      
      <h3 style="color: #000; font-size: 1.5rem; margin: 2rem 0 1rem 0; text-align: center;">Решаемые Проблемы</h3>
      
      <p class="chapter-text">
        До внедрения системы библиотека сталкивается с:
      </p>
      
      <ul class="analysis-card-list" style="text-align: left; max-width: 800px; margin: 0 auto;">
        <li><strong>Сложностями учета:</strong> неясно, какие экземпляры доступны, какие выданы, какие наиболее востребованы.</li>
        <li><strong>Потерей времени:</strong> ручная запись в журналах, постоянное физическое отслеживание местоположения экземпляров.</li>
        <li><strong>Рисками:</strong> потеря экземпляров, ошибки в учете выдачи, неудовлетворенность читателей из-за долгого поиска информации.</li>
     </ul>
   </div>
 </div>
</div>
